# Folder Logo

Folder ini berisi semua **logo / ikon utama** yang dipakai di Comfort Nook.
Kamu bisa **mengganti file di sini sesuka kamu** — selama nama filenya tetap
sama, kode tidak perlu diubah.

| File          | Dipakai di                         | Keterangan                    |
|---------------|------------------------------------|-------------------------------|
| `brand.png`   | HUD (pojok kiri atas), favicon ide | Logo utama "Comfort Nook"     |
| `music.png`   | Tombol Music di HUD                | Ikon untuk tombol pemutar musik |
| `comfort.png` | Tombol "Aku butuh ditemenin" Home  | Ikon pelukan/comfort          |

## Cara ganti
1. Siapkan gambar baru (PNG transparan recommended, ukuran 256×256 ke atas).
2. Rename ke nama yang sesuai tabel di atas.
3. Drop ke folder `src/assets/logos/` (replace yang lama).
4. Refresh preview — selesai!

> Tip: bisa juga `.gif` animasi. Tinggal rename jadi `brand.gif` dan ubah
> import di file kode dari `brand.png` → `brand.gif`.
